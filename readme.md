## Простой чат на socket.io ⚡

Реализованно:
  * Регистрация 
  * Авторизация
  * Поиск друга
  * Общение
  * Отправка фото

Stack ⚓
* React
* Socket.io
* Express
* TypeScript

> Вместо Redux реализовал store на React.context api.

--------

### команды: 
запуск клиента
```
client
```

запуск сервера
```
server
```

билд клиента
```
build:client
```

установка пакетов на клиент
```
install:client
```

установка пакетов на сервер
```
install:server
```

установка всех пакетов
```
install:all
```

старт для разработки
```
start
```
